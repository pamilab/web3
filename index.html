FMCW Ground Penetrating Radar Prototype

An advanced 2.4 GHz FMCW radar system that reliably detects underground objects with instantaneous signal processing and self-directed scanning.

This is a complete engineering solution — from concept to real-world performance.

Why This Radar Matters

Reliable Subsurface Detection

Produces clear radar data to locate buried objects and estimate depth without bulky hardware or high cost.

Portable and Easy to Deploy

Compact design built for field testing, research, infrastructure inspection, and disaster response.

Validated Results

Tested under controlled conditions with measurement errors below 2% for reliable confidence.

About the System

Low-cost, portable 2.4 GHz FMCW-GPR designed using accessible components to make subsurface sensing affordable and scalable.

Utilizes STFT signal processing and SDR digitization for real-time radargram visualization and autonomous scanning.

Built for practical research use and real-world engineering applications.

Building the Prototype

Concept & Planning

Defined limitations, selected FMCW architecture, and set requirements for an affordable, portable design.

Design & Simulation

Analyzed radar equations, chirp behavior, and depth–frequency relations before hardware implementation.

Hardware Integration

Assembled RF, controller, digitizer, and mechanical scanning platform into a stable prototype.
